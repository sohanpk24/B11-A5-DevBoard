Question-1: What are the different ways to select an element in the DOM?

যদি id থাকে, তাহলে getElementById() ইউজ করা হয়। একাধিক এলিমেন্ট সিলেক্ট করতে getElementsByClassName(), querySelectorAll() ব্যবহার করা হয়। CSS className দিয়ে এক্সেস করার জন্যে ব্যবহার করা হয়। querySelector() এবং querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML দিয়ে টেগসহ এক্সেস করার জন্যে ব্যবহার হয়। innerText টেগের ভিতরের টেক্স কে এক্সেস করার জন্যে ব্যবহার করা হয়। textContent কমেন্ট সহ সকল টেক্স কে এক্সেস করার জন্যে ব্যবহার করা হয়

Question-3: What is event delegation in the DOM?

অনেক গুলো elements কে evenListener না করে parent div এর মধ্যে event delegation

Question-4: What is event bubbling in the DOM?

child element থেকে parent তার উপরে যদি যদি grandParent div এভাবে এক লেভেল এক লেভেল করে উপুরে যাওার জন্যে event bubbling ব্যবহার হয়

Question-5: How do you create, add, and remove elements using JavaScript?

html element থেকে কিছু যোগ করা কিছু ডিলিট করার এবং নতুন html tag বাননোর জন্যে ব্যবহার করা হয় create, add, and remove elements
